# pieris.celldeath.brapa.QTLmapping

This repository contains the datasets and codes used in the study _"Genetic analysis reveals three novel QTLs underpinning 
a butterfly egg-induced hypersensitive response-like cell death in Brassica rapa"_ of Bassetti et al. 2021.

For deatils on files content and use of scripts, see "README_pieris.celldeath.brapa.qtl"
